## Goal

Full chain: **intent → entity → retrieve → Mini → validate**, with grounded mode that refuses answers without sources. Demo: cotton disease / pink bollworm with citations.

## Delivered

1. **`mini/inference/rag_wrap.py`** — wraps `advanced_rag` + mini-local lexical KB  
2. **`mini/inference/context.py`** — citation markers `[1]…` context pack  
3. **`mini/inference/generate_mini.py`** — Mini LM generation (SYSTEM_RAG)  
4. **`mini/inference/validate.py`** — grounding score + banned unsafe patterns  
5. **`mini/inference/fallback.py`** — template synthesizer on low confidence  
6. **`mini/inference/pipeline.py`** — end-to-end `run_infer`  
7. **`W-RAG`**, **`W-AGENT`** (light), **`W-INFER`** ready  
8. CLI `infer` / `rag` · API `GET/POST /api/lake/infer`  
9. Pipeline `sprint15`
