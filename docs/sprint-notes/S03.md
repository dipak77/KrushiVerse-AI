## Goal

Automated quality gate on every raw batch: validate → clean → dedup, with quarantine and quality reports.

## Delivered

1. **W-VALIDATE** — JSON/type checks per domain; invalid files copied to `data/lake/quarantine/`  
2. **W-CLEAN** — HTML/script strip, unicode NFKC, whitespace normalize → `data/lake/processed/`  
3. **W-DEDUP** — exact SHA + near-dup Jaccard on char 3-grams  
4. **W-QUALITY** — full pipeline + `QUALITY_LATEST.json` / `reports/quality_*.json`  
5. CLI `quality --execute`, pipeline `sprint3` / `quality`  
6. API `GET/POST /api/lake/quality`
