## Goal

Make Mini a product-facing synthesizer behind a feature flag, while specialized agents remain tool channels. Ship `POST /api/mini/chat` and a Streamlit Mini panel with citations.

2. **`app/llm/mini_bridge.py`** — `run_mini_chat` + agent-output notes
3. **`app/llm/generator.py`** — optional Mini synthesizer path
4. **`app/agents/planner.py`** — passes query/crop into synthesizer; returns `use_mini_llm` / `synthesizer`
5. **FastAPI**
   - `GET /api/mini/status`
   - `POST /api/mini/chat` (always Mini chain; not gated by flag)
6. **Streamlit** — “Mini LLM + Citations” tab + sidebar flag readout
