# Sprint 9 Notes — Domain Tokenizer v0.1 (W-TOKEN)

**Status:** Complete  
**Sprint:** S9  
**Feature phase:** FP-5  
**Date:** 2026-07-18  

## Goal

Agriculture-aware SentencePiece tokenizer v0.1 with 30–50k vocab, force-included domain symbols, and fertility checks on EN/MR agri probes.

## Delivered

1. **`mini/tokenizer/train.py`**
   - Domain corpus from taxonomy, processed lake, platform KB, QA synth, KG triples, combinatorial agri lines
   - Synthetic lexicon pad so BPE can reach ≥30k
   - Force-include crops / pests / fertilizers / districts / stages / schemes
   - BPE training default vocab **32000**
   - Baseline model without force-symbols for fertility comparison
   - Crop symbol hit-rate as primary domain fertility signal
2. **`W-TOKEN`** ready worker
3. CLI `token`, API `/api/lake/tokenizer`, pipeline `sprint9`
4. Artifacts under **`mini/tokenizer/v0.1/`**
   - `sp_agri.model` / `.vocab`
   - `sp_baseline.model` (comparison)
   - `manifest.json`, `TOKENIZER_LATEST.json`
5. Dependency: `sentencepiece>=0.2.0` in `requirements.txt`
