## Goal

Train the ~1M Mini decoder on agriculture text with packed contexts, report val perplexity improvement vs init, and save reproducible checkpoint `v0.2-base` (local only).

   - Context packing into fixed blocks + train/val split
2. **`mini/models/pretrain.py`**
   - Stream batches, AdamW, train loss curve
   - Val PPL at start/end + intermediate evals
   - Sample completion
   - Checkpoint `mini/models/v0.2-base/` (gitignored)
   - Seed reproducibility helper
3. **`W-PRETRAIN`** upgraded: `mode=domain|smoke|both`
4. CLI `pretrain --mode domain --steps 200 --seed 42`
5. API `POST /api/lake/pretrain?mode=domain&steps=200`
6. Pipeline `sprint11`
